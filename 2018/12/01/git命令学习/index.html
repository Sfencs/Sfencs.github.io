git命令学习

发表于 2018-12-01 | 分类于 learn

初始化名字email

$ git config –global user.name “Your Name”
$ git config –global user.email “email@example.com“

仓库初始化，变成git可管理的仓库

$ git init

向暂存区添加文件

$ git add

将暂存区内容提交到历史记录区

$ git commit -m

查看仓库状态

$ git status

比较状况，工作区与暂存区

$ git diff
$ git diff

比较状况，工作区与版本库

$ git diff HEAD –

比较状况，暂存区与版本库

$ git diff –cached

查看日志

$ git log
$ git log –pretty=oneline

版本回退

$ git reset –hard HEAD^
$ git reset –hard HEAD~3
$ git reset –hard 548b5

查看命令历史

$ git reflog

撤销对工作区文件的修改，还没有提交到暂存区

$ git checkout –

对暂存区的修改撤销

$ git reset HEAD

删除暂存区可工作区的文件

$ git rm

仅删除暂存区的文件

$ git rm –cached

与github仓库关联

$ git remote add origin git@github.com:Sfencs/learngit.git

第一次推送加关联master分支

$ git push -u origin master

之后的推送

$ git push origin master

克隆远程库

$ git clone https://github.com/Sfencs/learngit.git

创建分支

$ git branch

切换分支

$ git checkout

创建并切换分支

$ git checkout -b

查看分支

$ git branch

合并指定分支到当前分支

$ git merge

删除分支

$ git branch -d

查看分支合并

$ git log –graph –pretty=oneline –abbrev-commit
$ git log –graph

不使用fast forward方式合并，使用普通方式

$ git merge –no-ff -m “merge with no-ff”

储藏现场

$ git stash

查看储藏

$ git stash list

恢复现场

$ git stash apply

删除储藏

$ git stash drop

恢复现场并删除储藏

$ git stash pop

恢复指定储藏

$ git stash apply stash@{0}

强行删除没有被合并的分支

git branch -D

查看远程库的信息

$ git remote
$ git remote -v

把分支推送到远程库对应分支

$ git push origin dev

创建远程库的分支

$ git checkout -b dev origin/dev

指定本地分支与远程分支的链接

$ git branch –set-upstream-to=origin/dev dev

取回远程某个分支的更新，再与本地的指定分支合并

$ git pull

变基，把本地未push的分叉提交历史整理成直线

$ git rebase

多人协作的模式

首先，可以试图用git push origin 推送自己的修改；
如果推送失败，则因为远程分支比你的本地更新，需要先用git pull试图合并；
如果合并有冲突，则解决冲突，并在本地提交；
没有冲突或者解决掉冲突后，再用git push origin 推送就能成功！
如果git pull提示no tracking information，则说明本地分支和远程分支的链接关系没有创建，用命令git branch –set-upstream-to origin/。

打标签，默认在最新提交的版本

$ git tag v1.0

查看所有标签

$ git tag

按commit id 打标签

$ git tag v0.9 f52c633

查看标签信息想

$ git show

创建带有说明的标签

$ git tag -a v0.1 -m “version 0.1 released” 1094adb

推送标签到远程

$ git push origin v1.0

一次性推送所有未推送到远程的标签

$ git push origin –tags

本地删除标签

$ git tag -d v0.9

删除远程标签

$ git push origin :refs/tags/v0.9
