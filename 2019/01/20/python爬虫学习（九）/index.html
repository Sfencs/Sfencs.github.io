前言

Scrapy是基于Python的分布式爬虫框架，从现在起学习Scrapy框架，之前的android原生app爬虫，由于不知为什么uiautomator的python库使用总是不成功，所以暂时先鸽了，等有空看看appium。

Scrapy安装

  • Visual C++ Build Tools的安装，之前一直在用c和c++，应该不用再安了
  • pywin32，用pip装了下，发现已经有了
  • Twisted，是python中一个基于事件驱动的异步输入输出引擎，依赖于pywin32与Visual C++ Build，使用pip安装
  • 安装Scrapy，同样pip安装。
