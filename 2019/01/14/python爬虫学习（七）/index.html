python爬虫学习（七）

发表于 2019-01-14 | 分类于 爬虫

前言

通过network寻找通过异步加载方式显示的内容会非常麻烦，需要一个一个的查看，要想要迅速查找，就需要全局搜索所有请求返回的数据，这是就需要charles

Charles

Charles是一个跨平台的HTTP抓包工具，下载地址为https://www.charlesproxy.com/download/
安装打开后，基本的使用为：

扫把是清空
红色圆点是暂停
Filter中可输入域名进行过滤
Contents中上部分是与请求有关的部分，下部分是与返回有关的部分

接下来使用Chrome安装第三方代理插件SwitchyOmega，设置代理IP为127.0.0.1，端口号为8888，这样让Charles监控Chrome

当charles抓包以后，如果是windows系统，那么按下ctrl+F进入搜索，然后输入关键词find，便会找到存在该关键词的数据包。但是json中的中文是无法搜到的，需要在python中使用json.dumps转换成unicode码来搜索。

对于抓取HTTPS的数据包，会出现Method为CONNECT的请求，但又会全部失败，这时需要安装SSL证书：

点击菜单栏Help-SSL Proxying-Install Charles Root Certificate,然后点击安装证书-下一步，选择“将所有证书都放入下列存储”，点击浏览，选择受信任的根证书颁发机构，点击确定，之后的对话框也点是。安装好证书后，在菜单栏的Proxy-SSL Proxying Settings中，点击Add，在Host中输入*,在port中输入443。之后就可成功捕获HTTPS数据包。

之后遇到找不到的请求，从charles这里搜索就可以了

app与小程序爬虫

charles不仅不仅可用于搜索功能，还可以抓取手机的数据包，所以首先要把证书安装到手机上
在charles菜单中点击Help-SSL Proxying-Save Charles RootCertificate将证书保存下来然后发送到手机（android）上，然后在手机的wlan里的高级设置中找到安装证书，将刚发到手机上的证书安装，之后在当前连接的wifi设置中将代理改成charles的IP和端口号。
不知道什么原因，每次一设置完代理，手机的wifi就不能连接到网络了，试了好几次也没成功，就先暂时放弃了。。
